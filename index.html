Data Analyst Wokflow

Understanding Business Questions

Prior to initiating analysis, Data Analysts conduct comprehensive stakeholder engagements to define clear business objectives and establish precise analytical goals, ensuring alignment with desired outcomes.

Data Collection & Cleanning

Data Ingestion, Exploration and Collects relevant data from various sources, ensuring accuracy and completeness. Data cleaning involves removing duplicates, filling in missing values, and correcting errors.

Exploratory Data Analysis (EDA)

Uses various analytical techniques to find patterns, trends, and valuable insights. Techniques used can include descriptive statistics, data visualization, regression analysis, or machine learning techniques. Commonly used tools include SQL, Python (with libraries like Pandas and Scikit-learn), and R.

Data Analysis & Visualization

Provides the "what" and "why" behind the data, and data visualization provides a clear and compelling way to communicate those findings.

Interpretation and Reporting

Interprets results, compiles reports, and communicates findings to non-numerical audiences.
